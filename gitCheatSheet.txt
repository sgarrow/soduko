set remote repository on github:
git remote set-url origin https://github.com/sgarrow/soduko.git

push local repository to github:
git push -u origin main

add new file to local repository:
git add .\gitCheatSheet.txt

commit new file to local repository:
git commit .\gitCheatSheet.txt -m "initial commit"

git diff:
git diff

commit changes to existing file (add then commit):
git add .\gitCheatSheet.txt
git commit .\gitCheatSheet.txt -m "added new cheat"

To remove a file locally and from the repository:
 git rm -f myFile.py
